Understanding Andar Bahar

Andar Bahar is a traditional Indian betting game, which is incredibly simple to play. The game uses a standard deck of 52 cards, and the aim is to predict whether a card with the same face value as the game card will appear on Andar (inside) or Bahar (outside).
